# selection-of-kiosk-experience-zone-location
2021 데이턴십 해커톤 4회 서울 8조 : 서울시 디지털 취약계층을 위한 키오스크 체험존 입지 선정

## 분석 목적
키오스크 체험존을 최적의 입지에 배치하여 디지털 취약계층인 고령층의 정보격차를 해소하고 디지털 기기의 접근성 및 활용능력을 향상시키기 위함
